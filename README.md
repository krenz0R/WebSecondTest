# WebSecondTest
Решение второго этапа по Web программироанию. Реализовал главную страницу сайта Инжинириум МГТУ им.Н.Э. Баумана. Так же были созданны js анимации. Модальное окно по кнопке "записаться", слайдер курсов, анимация наведения на элементыв в разделе "Курсы "Инжинириум МГТУ им. Н.Э. Баумана".
P.S Некоторые картинки не получилось скачать с сайта не судите строго)

Cписок предложений по изменениям и доработкам сайта:
1. Не очень качественные картинки, логотипы желательно сделать в svg формате.
2. На слайдере по умолчанию кнопки для перелистывания показываются только при наведении. Не понимающие пользователи могут даже не навестись на этот блок и следовательно даже не пролистают предложенные другие курсы.
3. Под слайдерами есть иконки курсов. Тоже сразу даже я не понял что по ним можно нажимать и они перелистывают слайды. Так же они не привликают внимание, считаю что на них почти никто не нажимает.
4. В разделе  "Курсы "Инжинириум МГТУ им. Н.Э. Баумана" на главной странице курсы кликабельны, но переходит на страницу где написано "курс не найден". Вставить правильную ссылку или же убрать вовсе переход.
